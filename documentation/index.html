Thank you for purchasing. This documentation will help and guide you to use and customize this template.

NAVIGATION

TOP / BOTTOM NAVBAR

There are several option we can do with navbar by using navbar class. Below is the structure of navbar class.

class=" [position] [width] [color] [nav-alignment] [sticky-nav] "

[position]
class : top-bar or bottom-bar
This is a position of navbar. It can be at the top or bottom of the slider. This position is also depend on where we put the code of the navigation. To use top bar, put the navigation section (wrapped by header, see screenshot above) at the top of page (below preloader section). To use bottom bar, put the navigation section inside main slider (at the bottom of slider). See HTML Structure for the layout. For more example see index.html, index-topbar-color.html, index-bottombar-color.html and index-bottombar-transparent.html

[width]
class : fixed-width or fluid-width
This is the width of nav-wrapper. Inside this nav-wrapper, there are logo and navigation(menu).
Fixed width: 1170px and Fluid width:100%.

[color]
class : block-color or transparent
This is the color of navbar. Transparent will set the navbar background to no color, so it will blend to slider. Block Color will set a white background for the navbar. The color can be change easily by using skin css.

[nav-alignment]
class : nav-center or nav-left or nav-right
This is the alignment between logo and navigation. The navigation can be at left, right or center.
nav-center: logo and navigation align center.
nav-left: navigation at left side and logo at right side.
nav-right: navigation at right side and logo at left side.

[sticky-nav]
class : sticky-nav
Stick navbar at the top of window when we scroll down the page. To unstick or not use this, just remove this class.

 

Navigation/Menu
Inside Navbar we can find the navigation (menu). It uses <ul> and <li> structure. The navigation also provide additional class:

[first-child]
This class will remove default heart icon in front of the menu. Usually use only to first child.

[first-child-margin]
Set margin-left (default 20px). Usually use only to first child and if we use center alignment for the menu.

[last-child-margin]
Set margin-right (default 0). Usually use only to last child and if we use center alignment for the menu.

[split-margin]
Set margin-left (default 80px). This margin will be used to split navigation into right and left with the logo on center.

SIDENAV BAR

There are two nav used on sidebar page: Left/Right Navigation and Menu Icon. Left/Right Navigation will be the page menu and Menu Icon will be a link to open the Left/Right Navigation.

To change the position of the Navigation and Menu Icon, we need to change the class for each of them. See below screenshot for more detail about it.

The screenshot is the example of using right position for the navigation and menu icon. To change it to left position, we just need to change:

  • sb-right -> sb-left
  • menuicon-right -> menuicon-left
  • sb-toggle-right -> sb-toggle-left

For menu icon bar, we also can set the width to fixed-width (1170px) and fluid-width (100%).

SLIDER

STRUCTURE

Slider is divided into two main part: Slide Title and Slide Item.

Slide Title
Slide title is a main title for the slider. Every slide item will share the same title by using this slide title. See the example of using slide title in index.html and index-topbar-color.html.

Slide Item
Slide Item contains images/video, overlay, and caption. We can have more than one slide item (except video). Slide caption is different with Slide title, Slide caption is a caption/title for each slide item. See the example of using slide caption in index-title-animation.html.

Slider Height
This template provide two class for the height of the slider, fullscreen and fixed-height. Fullscreen will make the slider always fit with window size. And Fixed height will set the default height to 625px.

Flexslider
To use more than one images for slide item, we have to add flexslider class to main-slider id.

FADE / SLIDE

If we are using flexslider, we can use fade or slide effect for the slider.

To use fade effect, we need to include main-slider-fade.js

To use slide effect, we need to include main-slider-slide.js

See example on index.html and index-slide.html

IMAGE ANIMATION

We can also animate the slide item (images) for the slider with Zoom Effect or Left-Right Movement Effect. To use this effect, we need to add loop-animation class to each slide.

And we need to include main-slider-image-animation.js

See example on index-image-animation-v1.html and index-image-animation-v2.html

Note: This animation will need a high memory usage. And for slider with slide effect, this image animation is not supported.

TITLE ANIMATION

To create title animation, we should add use-animation class to slide caption. And then followed by type of animation (fadeIn, fadeOut, etc). Find more type of animation from this site.

And we need to include main-slider-title-animation.js

See example on index-title-animation.html

Note: For slider with slide effect, this title animation is not supported

TITLE & IMAGE ANIMATION

To create title and image animation, we need to include main-slider-image-title-animation.js. To add title animation see Title Animation, to add image animation see Image Animation.

See example on index-image-title-animation.html

Note: For slider with slide effect, this title animation is not supported

For play/pause button, we can change the icon easily by replace <i> tag class. There are also additional data to change the icon when we click the button.

Some elements of this template will animate when scrolling. To create an element animate when scrolling, just add animation class and type of animation (fadeIn, fadeOut, etc) to the element. Find more type of animation from this site.

There are also delay class to delay the animation:

PAGE LAYOUT

This template provide several body class to create padding/margin for our content wrapper.

Page Without Slider
Add "no-slider-page" class to body, this will create padding:140px 0 80px 0 for content wrapper.

Page With Slider
Add "slider-title-page" class to body, this will create padding:80px 0 for content wrapper.

Full Gallery Page
Add "full-gallery-page" class to body, this will create padding:100px 0 80px 0 for content wrapper.

Page With Outside Bottom Bar
Add "outside-bottom-bar-page" class to body, this will create margin-top:100px for content wrapper.

PAGE ELEMENT

IMAGE / VIDEO DIVIDER

To use Image Divider, create a div and add image-divider class. Then set the height of the image divider by adding fixed-height or auto-height class. By default fixed-height class will set the height to 700px, and auto-height will have a 100% height depend on the content in image divider. Then add style for background-image.

We also can add overlay by using divider-overlay class. See example on page.html

We can do the same if we want to use Video Divider but now we use video-divider class (not image-divider), the only different is image divider call image from background image and Video Divider call Video by using data-vide. See example on page.html. Note: Video Divider do not support parallax effect.
